## 1. 概念定義 (Concept)

**「家を建てる」のではなく、「インフラを接続する」。**

SUJ (SONA Universal Joint) は、SONAハビタット・システムにおける**「構造（Structure）」**と**「配管（Infrastructure）」**を統合した、アルミ押し出し材による汎用締結モジュールである。

本仕様は、以下の3要件を満たすよう設計されている：
1.  **Structural:** 3mパネルを強固に保持する「柱」であること。
2.  **Infrastructural:** 内部に電力・水・データを安全に通す「血管」であること。
3.  **Reversible:** 素人がレンチ1本で組立・分解・再利用が可能であること。

## 2. 形状と寸法 (Geometry & Dimensions)

### 2.1 プロファイル形状 (Profile)
基本形状は**「H型（H-Beam）」**を採用する。

```text
      [ PANEL A ]                   [ PANEL B ]
          ↓                             ↓
    +-----------+                 +-----------+
    |           |                 |           |
+---+           +-----------------+           +---+
|   |   (W)     |    (CORE)       |    (E)    |   |
| G |  Water    |   Structure     |   Power   | G |
|   |  Chamber  |   & Drainage    |   Data    |   |
+---+           +-----------------+           +---+
    |           |                 |           |
    +-----------+                 +-----------+
          ↑                             ↑
    (Gasket Seal)                 (Gasket Seal)
```

*   **(CORE):** 中央構造体。高剛性エリア。下水(Grey water)や構造ボルトを通す。
*   **(W):** Water Chamber。給水管（PEX管）を通すための独立区画。
*   **(E):** Energy Chamber。電力線・LANケーブルを通すための独立区画。
*   **(G):** Gasket Groove。防水パッキンを嵌め込む溝。

### 2.2 基本寸法 (Dimensions)
*   **全長 (Length):** `3,000mm` (トラック積載限界および居住性確保)
*   **フェイス幅 (Face Width):** `150mm` (断熱材入りパネルの標準厚に対応)
*   **材質 (Material):** アルミニウム合金 A6063-T6 (耐食性・加工性重視)

## 3. インフラ統合機能 (Integrated Utility)

柱そのものがライフラインとして機能する。

### 3.1 分離原則 (Separation Protocol)
漏電・汚染事故を防ぐため、物理的に区画を分離する。
*   **左翼 (W):** **水専用レーン**。万が一水漏れしても、右翼には浸水しない。
*   **右翼 (E):** **電気・通信専用レーン**。ノイズ対策のためアルミ隔壁でシールドされる。

### 3.2 プラグ＆プレイ (Plug & Play)
ジョイントの上下端には、SBCM標準コネクタ (**SUC**) との嵌合部を設ける。
*   柱を立てて接続するだけで、自動的に「1階から2階へ」「A室からB室へ」通電・通水する。

## 4. バリエーション (Variations)

H型を基本とし、拡張のための派生型を定義する。

| 型式 | 形状 | 用途 | 接続角度 |
| :--- | :--- | :--- | :--- |
| **SUJ-H** | **H型** | 直線連結 (壁の延長) | 180° |
| **SUJ-L** | **L型** | コーナー (四角い部屋) | 90° |
| **SUJ-T** | **T型** | 間仕切り接続 | 90° / 180° |
| **SUJ-X** | **十型** | 4部屋連結 (ハブ) | 90° x 4 |
| **SUJ-Hex** | **Y型** | ハニカム構造 / ドーム | 120° |

## 5. 施工プロセス (Assembly Process)

1.  **Set:** 地面にベースプレートを置き、SUJ（柱）を立てる。
2.  **Slide:** 左右から3m規格パネルをスライド挿入する。
3.  **Lock:** 側面のボルト穴から「圧縮ボルト」を締め込み、ガスケットを圧着させる（止水）。
4.  **Connect:** 柱の内部に配線・配管を通す（あるいはプリセットされたカセットを挿入する）。
